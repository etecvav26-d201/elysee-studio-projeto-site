# ✨ Élysée Studio

Projeto fictício desenvolvido para a disciplina do curso técnico de Informática para Internet — Turma 2°D.

O objetivo deste repositório é documentar e desenvolver um site moderno, responsivo e funcional para um salão de beleza fictício chamado **Élysée Studio**, aplicando conceitos de planejamento web, identidade visual, organização de projetos e desenvolvimento front-end com foco em experiência do usuário.

---

# 📌 Objetivo do Projeto

O projeto tem como finalidade criar um site de salão de beleza onde os usuários possam:

- visualizar serviços oferecidos;
- escolher procedimentos estéticos;
- consultar informações do salão;
- realizar agendamentos;
- acessar o site por dispositivos móveis de forma confortável e organizada.

Além disso, o projeto busca aplicar conhecimentos adquiridos durante o curso, como:

- HTML5;
- CSS3;
- JavaScript;
- Git e GitHub;
- responsividade;
- Mobile First;
- organização de projetos web;
- identidade visual.
