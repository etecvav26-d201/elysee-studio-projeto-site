# Conceito Mobile First

## Aplicação do Mobile First
O projeto será desenvolvido utilizando o conceito Mobile First, priorizando inicialmente dispositivos móveis e depois adaptando para telas maiores.

## Motivo da Escolha
A maioria dos usuários acessará o site utilizando celulares, tornando necessário um site rápido, leve e responsivo.

## Adaptação para Celulares
O site será adaptado utilizando:
- layouts flexíveis;
- imagens responsivas;
- botões maiores;
- menus simplificados.

## Cuidados com o Layout

### Menu
O menu será compacto e responsivo, podendo utilizar um menu hamburguer em telas pequenas.

### Imagens
As imagens serão otimizadas para evitar carregamento lento.

### Textos
Os textos terão:
- boa legibilidade;
- tamanho adequado;
- organização clara.

### Organização da Tela
Os conteúdos serão organizados em blocos verticais para facilitar a navegação em celulares.
