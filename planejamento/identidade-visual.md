# Identidade Visual

# ✨ Conceito da Marca

## Nome do Projeto
Élysée Studio

A identidade visual do Élysée Studio foi desenvolvida com foco em:
- elegância;
- minimalismo;
- sofisticação;
- modernidade.

O objetivo da marca é transmitir uma imagem premium e organizada, utilizando elementos visuais leves e refinados para representar beleza, cuidado pessoal e confiança.

---

# 🎨 Paleta de Cores

A identidade visual utiliza tons minimalistas com destaque para o amarelo dourado.

## Cores Principais

| Cor | Código HEX | Uso |
|---|---|---|
| Amarelo Dourado | #FFD600 | Destaques e identidade principal |
| Preto | #1A1A1A | Textos e contraste |
| Branco Gelo | #F7F7F7 | Fundo principal |
| Bege Claro | #FFF7E0 | Detalhes suaves |

## Sensação das Cores
- amarelo dourado: luxo, destaque e sofisticação;
- preto: elegância e contraste;
- branco: leveza e limpeza visual;
- bege claro: suavidade e conforto visual.
