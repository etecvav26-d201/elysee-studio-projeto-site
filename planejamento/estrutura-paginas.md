# Estrutura das Páginas

## Página Inicial (Home)
A página inicial apresentará:
- banner principal;
- apresentação do salão;
- destaques dos serviços;
- botão de agendamento;
- avaliações de clientes.

## Página de Serviços
Mostrará todos os serviços disponíveis:
- cortes;
- colorações;
- manicure;
- barba;
- hidratação;
- lavagem.

Cada serviço poderá conter:
- descrição;
- preço;
- imagem ilustrativa.

## Página de Agendamento
Permitirá:
- selecionar serviços;
- escolher data;
- escolher horário;
- preencher dados do cliente.

## Página Sobre
Apresentará:
- história do salão;
- missão;
- visão;
- informações da equipe.

## Página de Contato
Conterá:
- telefone;
- e-mail;
- endereço;
- redes sociais;
- formulário de contato.

## Navegação do Site
O site utilizará:
- menu responsivo;
- navegação simples;
- botões de acesso rápido;
- organização intuitiva.
